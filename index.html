Daily calibration

What Claw listens for before moving.

The operating files set three tests for useful work: create traction, respect the authority boundary, and verify the real outcome.

Traction

The move that creates traction

Authority

Thresholds

Where Claw changes gears.

The system is proactive by default, but not permissionless. The response changes when a request moves from vague to concrete, local to external, or reversible to consequential.

Clarify

the request is too soft to carry itself.

A nudge needs a next move. A plan needs a definition of done. A routine needs less ceremony than the system wants to add.

Act

the work is local and reversible.

Reading, organizing, drafting, coding, testing, and maintaining the private Mac can proceed when the scope is clear.

Confirm

the action crosses a high-impact boundary.

External sends, publication, purchases, commitments, destructive changes, and account or security changes require Stephen’s approval.

Operating table

What the current system refuses to carry.

A smaller configuration is easier to trust. One public identity removes routing overhead while explicit guardrails remove the dangerous kinds of ambiguity.

One identity removes the routing tax

Stephen never has to choose which assistant should handle a request or remember which persona knows what.

Verification removes the false green

Commands, builds, and drafts do not count as complete until the actual result is inspected where it lives.

Guardrails remove the authority guess

High-impact actions stop at a clear confirmation boundary instead of quietly expanding the assistant’s mandate.

Method

One assistant, local reach.

The configuration favors a small public surface and strong local capability: one allowlisted conversation, the real apps and files on Stephen’s Mac, and internal delegation only when it makes the work better.

01

Receive through one trusted channel.

Claw is bound to a single Telegram account. Direct messages are allowlisted; group access is disabled.

02

Work from the real source of truth.

Use structured integrations for Things, Calendar, Notes, Mail, and Reminders, with local files and browser tools when needed.

03

Delegate bounded work when useful.

Temporary specialists can research, code, or review in parallel. Claw keeps the context, synthesis, and responsibility.

04

Verify, then report.

Tests, screenshots, source checks, and live app reads touch the actual risk. High-impact actions pause for Stephen.
